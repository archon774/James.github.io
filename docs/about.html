Skynet Research Team

Learn More

Description

My research is focused on finalizing both the design and development of the radio astronomy portion of Skynet 2.0. Used by multiple universities across the nation, Skynet is the next generation robotic telescope network used to provide students with hands-on-access to optical and radio astronomy. Building on prior progress modeling the radio hardware and software, my work is centered around developing new user interfaces for submitting radio observations, ensuring the system can efficiently handle observation queues. Additionally, I will oversee managing the migration of select archival data from legacy servers to the new Skynet 2.0 infrastructure. Using the Angular web framework, PrimeNG, and Tailwind libraries the user interface will be implemented while the backend servers are built with Python and FastAPI with PostgreSQL as the database engine. The radio telescope server will be further developed in Python integrating both Astropy and the in-house Skylib package.

Skills & Impact

Python, Node.js, git, FastAPI, PostgreSQL, Angular, PrimeNG, Tailwind, Astropy, Docker, data migration, backend engineering, and frontend development.

ERIRA 2025

Learn More

Description

I participated in an immersive, intensive, one-week research experience at Green Bank Observatory during the summer of 2025. This highly selective program is designed to introduce students to hands-on radio astronomy in which I worked with GBO’s 40-foot telescope and fully automated 20-meter research grade telescope to conduct a series of independent research projects (more of which can be found on my projects page). Throughout the week I also conducted continuous observations mapping part of the galactic plane, explored stellar objects such as Pulsars, and analyzed strong radio sources such as Cassiopeia, Cygnus A, and the Crab Nebula. This program gave direct experience in configuring and operating scientific equipment, handling data acquisition software, and producing radio images of astrophysical regions.

Skills & Impact

Research collaboration, presentation skills, scientific communication, data processing, and data analysis.
